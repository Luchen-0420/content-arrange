### 1. 新建飞书表格

| 字段名     | 飞书字段类型 | 字段说明                                                                 |
|------------|---------------|--------------------------------------------------------------------------|
| 书籍名称   | 文本        | 书名                                               |
| 封面图     | 附件        | 封面图                                         |
| 类型       | 单选        | 属于什么类型                              |
| 作者       | 文本        | 作者                              |
| 简介       | 文本         | 书籍简介              |
| 进度      | 进度         |阅读进度        |
| 开始阅读   | 日期       | xxxx-xx-xx形式 |
| 结束阅读   | 日期       | xxxx-xx-xx形式 |
| 笔记链接     | 超链接   | 填写阅读笔记的超链接                              |
| 个人推荐指数   | 评分         | 个人觉得这篇书是否值得观看                   |
